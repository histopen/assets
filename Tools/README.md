## localization - toolloc
Localization files are json files. Do NOT edit json files, edit the googleSheet: 
https://docs.google.com/spreadsheets/d/122F4RZYbeBNl10tjWZYjimDzUazSMN1jyQaatOtxSR8

```bash
  "npm run loc"     to update all json files => public/locales  
```
```bash
  "npm run loc en"  to update english only  
```
  
language codes: en: English, fr: French, it: Italian, de: German, es: Spanish, nl: Dutch, pl: Polish, uk: Ukrainian, ru: Russian, sv: Swedish
